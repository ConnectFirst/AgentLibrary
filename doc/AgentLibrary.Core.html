AgentLibrary.Core

This is the constructor for the Library Object. Note that the constructor is also being attached to the context that the library was loaded in.
Properties:
Name Type Description
callbacks object Internal map of registered callback functions
_requests array Internal map of requests by message id, private property.
_queuedMsgs array Array of pending messages to be sent when socket reconnected
_isReconnect boolean Whether or not we are doing a reconnect for the socket
